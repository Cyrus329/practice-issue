# 根号公式显示优化说明

本版继续保留 1540 道正式题，并重点优化数学公式显示。

## 已处理

- `√(x²-1)` 会显示成带横线的根式。
- `√x`、`√2`、`√[(3x-2)/((5-2x)(x-1))]` 会按根号整体显示。
- `³√(...)`、`∛(...)` 会显示成立方根样式。
- `1/√(...)`、`3√2/2` 会尽量显示成分数形式。
- `dy/dx`、`dy/dt`、`dx/dt` 会显示成上下分式。
- 缓存版本号已更新，上传 GitHub 后会强制刷新旧缓存。

## 注意

这是离线 CSS/JS 渲染方案，不依赖外部网络。极复杂公式仍建议后续升级为 MathJax 或 KaTeX。
