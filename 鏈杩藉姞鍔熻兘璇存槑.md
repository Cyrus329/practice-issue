# 本次追加功能说明

本次按用户选择继续加入：5、6、7、17、19、20。

## 已加入

1. 熟练题自动隐藏
   - 连续做对 3 次后标记为“熟练”。
   - 开启“隐藏熟练题”后，这类题会从普通刷题列表中隐藏。
   - 状态筛选里新增“熟练题”，可以单独查看被隐藏的熟练题。

2. 每日任务量
   - 默认每日目标为 80 题。
   - 可以在学习分析面板中修改目标题数。
   - 每做对或做错一次都会计入今日完成量。

3. 倒计时刷题模式
   - 新增 25 分钟倒计时。
   - 顶部按钮可以快速开始/暂停。
   - 面板内可以重置为 25 分钟。

4. 英语选项解析强化
   - 英语选择题在“再看解析”层级中新增“英语选项辨析”。
   - 会按答案项与干扰项给出辅助排除提示。
   - 具体原因仍以原解析为准，避免强行编造不存在的逐项解析。

5. 数据备份提醒
   - 学习分析面板新增备份状态提示。
   - 有做题记录但从未备份，或超过 3 天未备份，会提示导出备份。
   - 导出时会记录本次备份日期。

6. 手机底部固定操作栏优化
   - 手机端“上一题 / 下一题 / 公式提示 / 查看答案 / 做对 / 做错 / 错题本”固定在底部。
   - 做题时不用反复上下滑找按钮。
