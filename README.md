# 专升本题库训练中心

这是一个独立题库训练页面，可以放在 GitHub Pages 上使用，也可以直接打开本文件夹里的 `index.html`。

## 上传到 GitHub

把本文件夹里的所有内容一起上传到仓库，例如放到 `question-bank/` 文件夹。

打开地址示例：

`https://你的用户名.github.io/仓库名/question-bank/index.html`

## 和错题本连接

把这个题库页面和原来的错题本放在同一个 GitHub Pages 仓库、同一个域名下使用。题目做错后点“加入错题本”，会写入现有错题本使用的本地数据。

## 导入题库格式

导入 JSON 可以是数组，也可以是：

```json
{
  "questions": [
    {
      "id": "MATH-001",
      "subject": "高数",
      "chapter": "函数与极限",
      "type": "选择题",
      "stem": "题目内容",
      "options": ["A. ...", "B. ...", "C. ...", "D. ..."],
      "answer": "B",
      "analysis": "解析过程",
      "difficulty": "2",
      "tags": ["函数"]
    }
  ]
}
```

图片题可以额外带：

```json
"images": ["question-images/example/example-001.jpg"]
```
