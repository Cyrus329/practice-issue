# v6 选中功能更新说明

已按用户选择加入：

1. 官方答案锁定机制：题目数据增加 answerAuthority / answerSource / analysisSource。
2. 答案来源显示：答案下方显示题源、答案来源、解析来源。
3. 原题图与解析图对照：支持 analysisImages，并给高数函数题加入官方解析页原图入口。
12. 数学函数解析清理：函数 54 题改用《数学基础精讲第一章第一节习题详解.pdf》的答案解析，删除模板废话。
13. 英语解析不再生成选项套话：英语选项辨析模块已停用。
17. 官方解析与 AI 整理分开：若存在 AI 整理，会单独显示，不和官方解析混在一起。
18. 短解析模式：默认显示短解析，点“展开完整解析”看全文。
25. 自动下一题：点击“做对了/做错了”后 0.5 秒自动跳下一题。
27. 图片放大：题图和解析图可点击放大。
28. 答案固定在题目下方：答案区直接出现在当前题下，不再跳到页面下方。
